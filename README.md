## 🌟 Overview
**Hozn - Real Estate** is a real estate website built with **React, Next.js, TypeScript** . The platform allows users to browse, list, and manage properties seamlessly.

## 🎯 Features
- 🔥 **Modern UI/UX** with smooth animations
- 🏠 **Property listing & management**
- 🏠 **BUY & Sell Property listing ** 
- 📊 **Admin Dashboard** for managing users & listings 
- 🎨 **Fully Responsive** on all devices

  
## 🛠️ Tech Stack

### 🌐 Frontend
- **React.js** (Framework: Next.js)
- **TypeScript**
- **Tailwind CSS & CSS** (for styling)
- **SCSS**
- **Framer Motion** (for animations)
- **Three.js**

## ⚙️ Installation & Setup

### 1️⃣ Install Node.js and npm
Ensure you have **Node.js** and **npm** installed. If not, install it from:
👉 [Download Node.js](https://nodejs.org/)

Check installation:
```sh
node -v
npm -v
```


### 3️⃣ Clone the Repository
```sh
git clone https://github.com/RaulRivas302/real-estate-system.git
cd Hozn-RealEstate-Fullstack
```

### 2️⃣ Install dependencies:
```sh
cd Hozn-RealEstate
npm install

```


### 4️⃣ Run the project:
```sh
# Start frontend
cd ../hozn-frontend

npx ts-node src/server.ts
or
npm run dev
```

```
